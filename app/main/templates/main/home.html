Django Forum Templates

Voici l'ensemble des templates que vous pouvez utiliser lors de la réalisation du projet.

Title Description
Login Le template de la page de connexion.
Register Le template de la page de création de compte.
Reset PWD Form Le template de la page de demande de changement de mot de passe.
Reset PWD Done Le template de la page de confirmation de demande de changement de mot de passe.
Reset PWD Confirm Le template de la page de changement de mot de passe.
Reset PWD Complete Le template de la page de confirmation de changement de mot de passe.
Profil Le template de la page profile.
Topic List Le template de la page principale du forum listant l'ensemble des Topics créés.
Topic Detail Le template de la page détail d'un Topic. On y retrouve l'ensemble des messages liés à ce topic, ainsi qu'un formulaire pour y ajouter un nouveau message.
Topic Create Le template de la page de création d'un nouveau Topic.
React Le template de base pour intégrer un composant ReactJs dans une vue Django.
